# Roulette Simulator

A lightweight browser game where you can:

- Spin an animated roulette wheel
- Place multiple bets on a realistic roulette-style betting board
- Earn (or lose) tokens based on outcomes
- Track detailed roulette stats and recent roll history
- Buy upgrades to improve your long-term gains

Project structure:

- `index.html` - markup
- `styles.css` - styling
- `app.js` - roulette game logic and UI behavior

## How to play

1. Open `index.html` in a browser.
2. Set your chip amount.
3. Click betting spots on the table to place chips.
4. Click **Spin** to play.
5. Review stats/history and use earned tokens to buy upgrades.
