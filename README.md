# Roulette Simulator+

A modernized browser roulette game with progression systems, game modes, challenges, cosmetics, and social features.

## Highlights

- **Core loop upgrades**: session missions, milestone rewards every 10 spins, and prestige resets with permanent growth.
- **Progression**: player XP/level, seasonal XP/levels, achievements, daily login streak rewards.
- **Challenges**: daily + weekly challenge tracking with meaningful token/gem/XP rewards.
- **Modes and modifiers**: Classic, High-Risk, Chaos, Speed + rotating per-spin modifiers.
- **More bet depth**: combo bets, streak bets, optional risk token that doubles swing.
- **Strategy tools**: save/load betting presets, pre-spin power choices, offense/economy/safety upgrade branches.
- **Retention features**: rotating event banner and reward boosts.
- **Cosmetics/shop**: unlock themes, wheel skins, and trails using gems.
- **Social hooks**: local leaderboard, shareable run summary text, optional challenge seed for deterministic runs.
- **Onboarding & accessibility**: tutorial dialog, colorblind mode, reduced motion mode, mobile-first controls.

## Project structure

- `index.html` – app markup and feature panels
- `styles.css` – layout, themes, accessibility and responsive styling
- `app.js` – game logic, progression, rewards, modes, social systems

## How to run

1. Open `index.html` in a browser.
2. Set chip amount and place bets.
3. Choose mode/power, optionally set challenge seed.
4. Spin and progress through missions/challenges/achievements.
5. Upgrade branches, unlock cosmetics, submit/share runs, and prestige for long-term scaling.

## Validation performed

- `node --check app.js`
