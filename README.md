# Roulette Simulator+

A modernized browser roulette game with progression systems, realistic table variants/rules, advanced bet systems, analytics, and social/tournament features.

## Highlights

- **Core loop upgrades**: session missions, milestone rewards every 10 spins, and prestige resets with permanent growth.
- **Progression**: player XP/level, seasonal XP/levels, achievements, daily login streak rewards.
- **Challenges**: daily + weekly challenge tracking with meaningful token/gem/XP rewards.
- **Authentic tables**: European, French, and American variants (with 00), house-edge display, table min/max and no-more-bets flow.
- **Modes and modifiers**: Classic, High-Risk, Chaos, Speed + rotating per-spin modifiers and event boosts.
- **More bet depth**: combo bets, streak bets, split/street/corner/six-line, neighbors/finale/call bundles, optional risk token.
- **Strategy tools**: presets + multi-slot strategy profiles, pre-spin powers, rerollable mission pools, optional auto-bet systems with safety stops.
- **Retention features**: rotating event banner and reward boosts.
- **Cosmetics/shop**: unlock themes, wheel skins, and trails using gems.
- **Analytics & simulation**: EV/volatility/drawdown snapshots, bankroll curve, hot/cold summary, quick spin simulation, and seed browser.
- **Social hooks**: category leaderboard, tournament mode (fixed bankroll/seed), replay links, ghost target, and run summary sharing.
- **Onboarding & accessibility**: tutorial dialog, onboarding paths, colorblind mode, high contrast mode, reduced motion, keyboard-accessible bet spots.

## Project structure

- `index.html` – app markup and feature panels
- `styles.css` – layout, themes, accessibility and responsive styling
- `app.js` – game logic, progression, rewards, modes, social systems

## How to run

1. Open `index.html` in a browser.
2. Set chip amount and place bets.
3. Choose table variant/rules, mode/power, optional seed/dealer settings.
4. Build advanced bets or strategy profile, optionally enable auto-bet safeguards.
5. Spin and progress through rotating missions/challenges/achievements/season track.
6. Analyze outcomes via analytics/simulation, submit/share/replay runs, and prestige for long-term scaling.
